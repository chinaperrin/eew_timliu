Data List_READ ME

The files in this folder ("data lists") are lists of text files ("singles") containing certain subsets of data. Each single is a list of a specific piece of data for all the stations that triggered for some set of events. Below is an explanation of each data list and which directory the files it refers to are in.

ba_5.5.txt
Back azimuth estimates of events greater than 5.5. Each single is a different type of estimation technique run at different times following p wave arrival.
